A large-scale RPG-style Minecraft mod built for Minecraft 1.20.1 (Forge).

Immortal RPG introduces a new combat system, skill mechanics, custom entities, advanced rendering effects, and a progression-based RPG experience inside Minecraft.

Features
Advanced Combat System
-Custom weapon types
-Active & passive skills
-Cooldown-based abilities
-Custom damage logic
-Visual damage numbers

Skill System
-Modular skill framework
-Server-synced ability logic
-Keybinding-based casting
-Custom particles & animations

Custom Entities
-Boss-level enemies
-Training dummy
-Special AI behavior
-Unique combat mechanics

Visual Effects
-Custom particle engine
-Energy beams
-Pulsing light effects
-Advanced tooltip rendering
-GeckoLib animation support

Custom Systems
-RPG progression mechanics
-Attribute scaling
-Custom GUI screens

Dependencies
-Minecraft 1.20.1
-Forge 47.x+
-GeckoLib
